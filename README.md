# PyAsteroids
An Original Atari Asteroids replica made by myself using Pygame.

![asteroids](https://user-images.githubusercontent.com/63219861/120959577-116e6980-c728-11eb-949a-f975d6854bba.gif)

<h2>What is the game about?</h2>
You control a starship in the middle of the space vacuum and you need to survive shooting asteroids and saucers that can destroy you.

<h2>Installation</h2>
- Install Python 3.x version here: https://www.python.org/downloads/<br>- Install Pygame by running the following command on terminal: pip install pygame

<h2>Running the game</h2>
Open your terminal and run the following command: python main.py

Hope you enjoy!
